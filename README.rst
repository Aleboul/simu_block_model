Random generation of copulas with Python
----------------------------------------

**CoPY** is a Python module to simulate copula.

Installation
------------

Dependencies
~~~~~~~~~~~~

CoPY requires :

- Python (>= 3.7)
- NumPY (>= 1.14.6)
- SciPY (>= 1.1.0)

Source code
~~~~~~~~~~~

You can check the latest sources with the command::

  git clone https://github.com/Aleboul/COPPY/
